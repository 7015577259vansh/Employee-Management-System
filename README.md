# Employee Management System

A desktop-based Employee Management System developed using Java Swing and MySQL. The application allows users to manage employee records through a simple and user-friendly graphical interface.

## Features

- Secure Login Authentication
- Splash Screen
- Dashboard
- Add Employee
- View Employee Details
- Update Employee Details
- Remove Employee
- Auto-generated Employee ID
- MySQL Database Integration using JDBC

## Tech Stack

- Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

## Project Structure

```
Employee-Management-System/
│── src/
│── bin/
│── .classpath
│── .project
```

## How to Run

1. Clone the repository.
2. Import the project into Eclipse.
3. Create the required MySQL database.
4. Update the database username and password in `conn.java`.
5. Run `Splash.java` or `Main_class.java`.
